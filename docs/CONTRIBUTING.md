# 🤝 Contributing Guide

Thank you for your interest in contributing to the **HMJBI Web Profile** project.

---

## 📌 How to Contribute

1. Fork the repository (if external contributor).
2. Create a new branch from `develop`.
3. Work on your assigned task.
4. Commit your changes using the correct format.
5. Submit a Pull Request (PR) for review.

---

## 🌿 Branch Naming

Use the following format:

```
feature/feature-name
fix/bug-name
docs/update-name
```

**Examples:**

* `feature/hero-section`
* `fix/navbar-mobile`
* `docs/readme-update`

---

## 📝 Commit Message Format

Use clear and consistent commit messages:

```
feat: add new feature
fix: fix bug issue
style: update styling
docs: update documentation
```

---

## 🎨 Coding Standards

* Use **TypeScript**
* Use **Tailwind CSS** for styling
* Keep components **reusable**
* Use **PascalCase** for component names
* Write clean and readable code

---

## 🔍 Pull Request Rules

Before submitting a PR:

* Make sure the project runs without errors
* Check responsive layout
* Ensure code follows project standards
* Provide a short description of your changes

---

## 📞 Need Help?

If you have questions, contact the project maintainer or create an issue in the repository.

---

Thank you for contributing 🚀
